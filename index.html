Triggeraid

Camera/Flash Trigger with I2C LCD, RTC, Light Sensor, Time Lapse Mode, External Sensors etc.

Download this project as a .zip file Download this project as a tar.gz file 
                                TriggerAid
                                ----------
                         The Camera/Flash Trigger!

Libraries required:

Wire - Arduino included Library LiquidCrystal_I2C - http://www.xs4all.nl/~hmario/arduino/LiquidCrystal_I2C/LiquidCrystal_I2C.zip DS1307RTC - http://code.google.com/p/arduino-time/source/browse/#svn%2Ftrunk%2FDS1307RTC Time - Arduino included Library MemoryFree - http://playground.arduino.cc/Code/AvailableMemory EEPROM - Arduino included Library multiCameraIrControl - http://sebastian.setz.name/arduino/my-libraries/multi-camera-ir-control/

With Built-in Light Sensor, IR LED for wireless camera triggering, two (2) optocouplers (for separate focus & shutter connectivity or 2 separate flash units).

Modes: -> Light Trigger -> Time Lapse (0-360 secs) -> External Trigger (for connecting (digital-output) sound/light/etc modules) -> Bulb mode (1-999 seconds) -> Fully customizable
